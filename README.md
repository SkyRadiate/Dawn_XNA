#Dawn Game Engine#
##黎明游戏引擎##
###遵循GPL v2协议，开发自XNA，使用C\# ###

* Dawn Game Engine使用了FMOD
* 请从[Dawn Game Engine Data][DGEData]克隆项目数据文件。

[DGEData]:https://git.oschina.net/SkyZH/Dawn_XNA_Data

***

* **如果你是其中之一**
* 1.会使用RPG Maker的Ruby Game Script System
* 2.或能够运用C/C++编写控制台程序
* 3.或会使用C#编写窗体应用程序
* 4.或对编写程序有浓厚兴趣 :-)
* 你可以快速地入手结构类似RGSS，
* 使用C#语言
* 更加强大的DGE!
